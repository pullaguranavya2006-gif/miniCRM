# 🚀 Client Lead Management System (Mini CRM)

## 📌 Project Description

The Client Lead Management System (Mini CRM) is a Full Stack Web Application developed to manage client leads efficiently. This project allows users to add, view, update, and delete client leads while tracking their status and maintaining important notes.

The application helps businesses organize customer information, monitor lead progress, and improve lead conversion management through a simple and user-friendly interface.

This project was completed as **Task 2** of the **Future Interns Full Stack Web Development Internship Program**.

# 🔄 Application Workflow

### Step 1

User enters lead details:

- Client Name
- Email Address
- Lead Source
- Lead Status
- Notes

### Step 2

The information is submitted through the web interface.

### Step 3

Express.js API receives the request.

### Step 4

MongoDB stores the lead information.

### Step 5

The dashboard automatically updates the lead statistics.

### Step 6

Users can view and manage all lead records.

# 📊 Lead Information Fields

| Field | Description |
|---------|------------|
| Name | Client Name |

# 🔗 API Endpoints

## Get All Leads

```http
GET /api/leads
```

## Add New Lead

```http
POST /api/leads
```

## Update Lead

```http
PUT /api/leads/:id
```

## Delete Lead

```http
DELETE /api/leads/:id
```
